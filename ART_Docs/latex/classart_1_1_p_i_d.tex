\doxysubsection{Detailed Description}
A general use \doxylink{classart_1_1_p_i_d}{PID} class. 

A \doxylink{classart_1_1_p_i_d}{PID} controller class. The class bundles all the functionality of a \doxylink{classart_1_1_p_i_d}{PID} controller into a single class so it can simply be fed inputs and output values back.

Most often, the \doxylink{classart_1_1_p_i_d}{PID} class will be used like this\+:

Calling this method updates several of the is the only was to update several of the member variables, so they will remain constant until \doxylink{classart_1_1_p_i_d_a6a918bed343b8650e62c6da061b2ae04}{calculate(double error)} is called.
