\doxysubsection{Detailed Description}
A Utility 2D Vector class. 

The \doxylink{structart_1_1_vec2}{Vec2} class aims to provide a better way to store coordinates, travel and displacement in a 2d space. \doxylink{structart_1_1_vec2}{Vec2} objects store thier x and y components, but has methods to access thier direction and magnitude as well. Vectors can be added together, scaled or multiplied by each other to find the dot product.

The components\textquotesingle{} units can be whatever you want, but the direction will always be in radians, as the math works best that way. The \doxylink{classart_1_1_angle}{Angle} Unit Utility Class defaults to Radians as a result, but as of right now, using the \doxylink{classart_1_1_angle}{Angle} class is optional.
